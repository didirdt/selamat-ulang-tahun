# Selamat Ulang Tahun

Project buat kamu yang mau ngasih hadiah ulang tahun simple website.

## Run Locally

Clone the project

```bash
  git clone git@github.com:didirdt/selamat-ulang-tahun.git
```

Go to the project directory

```bash
  cd selamat-ulang-tahun
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run server-node
```
